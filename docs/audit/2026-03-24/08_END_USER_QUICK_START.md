# End-User Quick Start (Mobile)

Updated: 2026-03-24

## 1) Start Here

1. Open app and sign in.
2. Confirm Dashboard loads.
3. Follow this daily order:
   - Dashboard
   - Pipeline
   - Calendar
   - Contact detail updates
   - Documents

## 2) Daily Mobile Flow

1. `Dashboard`
- Review today’s priorities and stage counts.

2. `Pipeline`
- Update customer stage as progress changes.

3. `Calendar`
- Create next appointments.
- Complete inspections when done.

4. `Contact Detail`
- Add notes, verify status/progress, check financial and docs tabs.

5. `Documents`
- Upload/open project docs and signed docs.

## 3) Most-Used Mobile Screens

- `Dashboard`: overview
- `Pipeline`: stage management
- `Calendar`: scheduling
- `Contact Detail`: customer hub
- `Documents`: file management
- `More/Settings`: profile/company settings

## 4) Quick Task Recipes

### Schedule Next Step
1. Open customer or Calendar.
2. Add appointment with date/time.
3. Save and verify on Calendar.

### Update Customer Status
1. Open Pipeline.
2. Move customer to correct stage.
3. Verify in Contact Detail.

### Upload and View Document
1. Open customer Documents or global Documents.
2. Upload file.
3. Tap to open/view.

### Update Avatar/Logo
1. Open Settings.
2. Upload image.
3. Crop and save.

## 5) Quick Troubleshooting

If something fails:
1. Retry once after refreshing screen.
2. Confirm required fields (date/time/customer).
3. Capture screenshot and note:
- customer
- action attempted
- expected result
- actual result
